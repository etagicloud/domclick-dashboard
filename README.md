# domclick-dashboard

Static HTML dashboard for the Domclick agency rating mini-DB.

- Source data: `D:\domclick\domclick.db` (SQLite)
- Local dashboard: `D:\domclick\dashboard.html` (generated by `python D:\domclick\domclick.py dashboard`)
- Deployed at: https://rating.megorov.com

## Deploy

Coolify watches `main` branch. After regenerating the dashboard locally:

```bash
cd D:\domclick
python domclick.py rebuild         # regenerates xlsx + dashboard.html
copy dashboard.html C:\beget\domclick-dashboard\
cd C:\beget\domclick-dashboard
git add dashboard.html
git commit -m "update: <YYYY-MM>"
git push
```

Coolify rebuilds the nginx image and rolls out (~60 sec).

## Dockerfile

Minimal nginx:alpine that serves dashboard.html as `/index.html`. Port 80 inside the container, Traefik provides HTTPS.
